# Personal Portfolio Website

This is a fully responsive personal portfolio website built using HTML and CSS.  
The design includes a header with navigation, a hero section, skills showcase, and a footer with social links.  
The website is optimized for desktop, tablet, and mobile layouts.



## Features

. Responsive layout (desktop, tablet, mobile)
. Hamburger menu on mobile
. Smooth scrolling navigation
. Animated hero section
. Skill cards with hover effects
. Clean and modern UI using the Poppins font
. No JavaScript (only HTML + CSS)

## AI Tool Used
I used Lovable.dev to generate the initial structure and basic layout of the portfolio website.
