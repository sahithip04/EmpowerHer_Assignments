# Enhanced Todo List

## Features
- Add tasks to the list
- Mark tasks as completed (strikethrough effect)
- Remove tasks individually
- Real-time search bar to filter tasks
- All tasks are stored in localStorage as a JSON array
- Each task has:
  - A unique ID
  - Task text
  - Completion status

## How to Use
1. Type a task in the input field and click **"Add Task"**.
2. Click on a task to mark it as **completed** or **uncompleted**.
3. Click the **"Delete"** next to a task to delete it.
4. Use the **search bar** to filter and view only matching tasks.
5. Tasks automatically save in your browserâ€™s localStorage.
6. Refreshing the page will keep all tasks intact.
